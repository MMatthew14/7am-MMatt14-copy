Instructions for iOS

On iOS devices, you need to install 7am as an app first before you can receive push notifications. Follow these steps:

  1. Open the share menu in Safari (the share button is in the middle, below the address bar)
  2. Swipe up, then tap "Add to Home Screen"
  3. Tap "Add" on the top right hand corner
  4. Tap on the newly added 7am icon.

You should now be able to subscribe to updates and receive them via push notifications.

Start using 7am.
